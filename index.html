Semântica

É o significado, o sentido e a interpretação das Tags.

Além de extruturar os sites, o HTML, hoje em dia, é usado para dar sentido usando as Tags semânticas.

Tags Semânticas

HEADER

A Tag header é usada para a construção de cabeçalho.

NAV

Já a Tag usada para a construção de um menu é a nav. Normalmente dentro da Tag nav, temos uma Tag de lista não ordenada ul.

  • Home
  • Sobre
  • Portifólio
  • Contato

MAIN

A Tag main é responsável por abrigar o conteúdo principal do site e de maior relevância. Ela abriga outras Tags semânticas como a Tag section

SECTION

A Tag section determina as seções do site. Dentro da Tag section podemos colocar um cabeçalho, ou seja, podemos colocar dentro da Tag section uma Tag header.

FOOTER

É a Tag footer a responsável pelo rodapé do site. Também podemos colocar uma Tag footer dentro de uma section.
